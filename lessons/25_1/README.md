# Подключение из MongoDB Compass

1. Откройте MongoDB Compass.
2. Нажмите «Connect» → «New Connection».
3. В строке подключения укажите:

```mongodb://admin:adminpassword@localhost:27017/?authSource=admin```

Либо используйте вкладку «Fill in connection fields individually» и заполните:

Host: localhost

Port: 27017

Username: admin

Password: adminpassword

Auth Database: admin

4. Нажмите «Connect».
