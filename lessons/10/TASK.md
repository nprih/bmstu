Проект 1

Вы пишите консольное приложение на Го
Приложение для библиотекарей
Объект: Книга. У книги есть: Название, автор, Флаг-наличие
Приложение д.б реализовано в виде бесконечного цикла с выводом меню выбора действия.
Действия:
1. Вывести весь "арсенал" библиотеки (все книги) КРАСИВО
2. Вывести только доступные/недоступные книги (выбор пользователя)
3. Вывести только книги, отсортированные по автору (автора вводит пользователь)
4. Взять книгу себе (изменить флаг) Предусмотреть невозможность взять недоступную книгу

ЗЫ заранее добавить несколько книг
