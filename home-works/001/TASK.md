# Домашнее задание №1 (глава 1.3)

## Задача 1
Необходимо оформить файл типа .md (MarkDown), в котором указываются основные команды и инструкции для работы с системой контроля версий git. Например: git init - команда инициализации репозитория; git add . - добавить все папки текущей директории в отслеживание изменений; и т.д. Этот файл необходимо выложить на github и прикрепить ссылку для просмотра.

## Задача 2
Написать программу на ЯП Go. Условие: На вход программе подаются 5 натуральных чисел. Задача программы - считать эти числа, отсортировать по убыванию и вывести результат сортировки на экран. Также вывести на экран другие параметры считанной последовательности в формате: "Самое большое число: {число}", "Самое маленькое число: {число}", "Среднее арифметическое: {число}"

## Пример работы программы:
выход: 5 2 4 3 1

выход:  
"Отсортированные элементы: 5 4 3 2 1  
Самое большое число: 5  
Самое маленькое число: 1  
Среднее-арифметическое: 3
"

Необходимо выложить программу на github и отправить ссылку для просмотра или отправить файл программы.
