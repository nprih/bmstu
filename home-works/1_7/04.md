1. Что такое frontend-разработка и чем она отличается от backend-разработки?
   - это создание внешнего интерфейса веб-сайтов и приложений.

2. Какие основные технологии используются для создания интерфейса веб-сайта или приложения?
   -  HTML, CSS, JS

3. Можешь ли ты назвать популярные языки программирования или инструменты, которые используют frontend-разработчики?
   - HTML, CSS, JS/TS (Jquery, Angular, React, Vue)
   
4. Что такое HTML, CSS и JavaScript, и какую роль они играют в создании веб-страниц?
   - HTML - язык разметки, отвечает за расположение элементов
   - CSS - каскадные таблицы стилей, описывают свойства элементов HTML
   - JS - добавляет интерактив на страницы
   
5. Какие задачи обычно решает frontend-разработчик при создании сайта или приложения?
   - Верстка, настройка стилей, настройка интерактива, SEO, настройка работы с API, адаптив.

6. Что такое адаптивный дизайн и почему он важен для frontend-разработки?
   - необходим для корректного отображения на экранах разного разрешения
   
7. Объяснить, что такое пользовательский интерфейс (UI) и пользовательский опыт (UX)?
   - UI - это всё, что пользователь видит на экране и с чем непосредственно взаимодействует: кнопки, меню, 
   поля ввода, иконки, списки, анимации и другие визуальные и интерактивные компоненты.
   - UX - это совокупность всех взаимодействий, эмоций и впечатлений, которые получает человек при использовании
   продукта, системы или услуги.

8. Какие современные фреймворки или библиотеки используются в frontend-разработке (например, React, Angular, Vue)?
   - React, Angular, Vue

9. Чем отличается статическая страница от динамической? Как это связано с работой frontend-разработчика?
   - статическая страница, все данные хранятся в коде HTML
   - динамическая страница, получает данные с backend и расставляет в нужные места HTML

10. Какие навыки или знания считаются обязательными для успешной работы в роли frontend-разработчика?
    - HTML, CSS, JS, React, Angular, Vue, понимание работы API
